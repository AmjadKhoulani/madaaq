# 🚀 منصة **MadaaQ** - نظام إدارة ISP المتكامل

## 📌 نظرة عامة
**MadaaQ** هي منصة سحابية متطورة لإدارة شركات مزودي خدمات الإنترنت (ISP) بشكل كليّ، مصممة خصيصاً للسوق العربي. توفر المنصة تكاملاً كاملاً مع MikroTik، إدارة العملاء، المحاسبة، والمراقبة الحيّة للشبكة.

---

## ✨ الميزات الرئيسية

### 1. 🌐 **إدارة الشبكة الكاملة**

#### إدارة MikroTik المتكاملة
- ✅ **تكامل API كامل** مع جميع أجهزة MikroTik
- ✅ **Auto-Sync** تلقائي للبيانات من/إلى الأجهزة
- ✅ **Terminal متقدم** لتنفيذ أوامر RouterOS مباشرة
- ✅ **Setup Scripts** تُولّد تلقائياً لكل جهاز
- ✅ إدارة **Servers** و**Routers** منفصلة
- ✅ استيراد المستخدمين والباقات من MikroTik

#### إدارة الباقات والخدمات
- 📦 **باقات Hotspot** كاملة مع User Profiles
- 📦 **باقات Broadband (PPPoE)** مع PPP Profiles
- 📦 إنشاء وتعديل الباقات بدون لمس الراوتر
- 📦 تفعيل/تعطيل الباقات بنقرة واحدة
- 📦 **تسعير مرن** مع أنواع اشتراكات متعددة

#### مراقبة الشبكة الحيّة
- 📊 **Dashboard حيّ** بإحصائيات real-time
- 📊 مراقبة **حالة الأجهزة** (Online/Offline)
- 📊 عرض **الجلسات النشطة** (Active Sessions)
- 📊 **التحكم بالسرعات** (Queue Management)
- 📊 **استهلاك البيانات** لكل عميل
- 📊 **Network Topology** تفاعلية لعرض بنية الشبكة
- 📊 **أكثر المواقع زيارة** من DNS Cache

### 2. 👥 **إدارة العملاء (CRM)**

#### نظام CRM متطور
- 👤 **قاعدة بيانات عملاء** شاملة
- 👤 **ملفات تعريف كاملة** لكل عميل
- 👤 **تتبع النشاطات** والملاحظات
- 👤 **حالات الاشتراك** (نشط، معلق، منتهي، محظور)
- 👤 **معلومات الاتصال** والموقع الجغرافي
- 👤 **ربط مع الأبراج** والسيرفرات

#### بوابة عملاء مخصصة (Client Portal)
- 🌟 **Subdomain مخصص** لكل فيندور (vendor.madaaq.com)
- 🌟 عرض **معلومات الحساب** والاستهلاك
- 🌟 **الفواتير** والمدفوعات
- 🌟 **تجديد الاشتراك** أونلاين
- 🌟 تطبيق موبايل (iOS/Android) قيد التطوير

### 3. 🎫 **إدارة كروت الهوت سبوت (Vouchers)**
- ✨ **توليد كروت جماعي** (Bulk Generation) - حتى 100 كرت مرة واحدة
- ✨ **معاينة مباشرة** لتصميم الكرت قبل الطباعة
- ✨ **صفحة طباعة احترافية** جاهزة للبيع
- ✨ **مراقبة الكروت المطبوعة** مع فلاتر وإحصائيات
- ✨ **إعادة طباعة** أي كرت سابق
- ✨ **بحث وفلترة** حسب الحالة، الباقة، التاريخ
- ✨ **تكامل تلقائي** مع MikroTik عند التوليد
